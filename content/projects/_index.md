## Портал документации Геомикс 2.0

Разработка и развитие системы пользовательской документации для инженерного ПО. Целевая аудитория документации: инженеры буровзрывных работ, маркшейдеры.

>Документация использовалась в пресейлах и хорошо себя показала: потенциальные пользователи прошли пользовательский путь (построили буровзрывной блок, расставили скважины, подготовили документацию), что укрепило их лояльность к продукту и подтвердило гипотезы продуктовой команды о необходимых бизнес-задачах, которые продукт должен помогать выполнять и автоматизировать. 

→ [Документация Геомикс 2.0](https://gmx2-doc.geomix.ru/)

### Моя роль

- Выявление требований к документации: когда пришла на проект, в команде не было представления, какой должна быть документация. Я это представление собрала воедино.
- Проектирование структуры документации с нуля.
- Организация информационной архитектуры.
- Подготовка контента по принципам Docs-as-Code.
- Настройка процессов разработки документации.

### Инструменты и подходы

- Docusaurus.
- Markdown.
- Git-based workflow.
- Docs-as-Code.
- [Diátaxis](https://diataxis.fr/).

### Результат

- Создана масштабируемая структура документации.
- Документация переведена в формат, удобный для совместной разработки.
- Подготовлена база для дальнейшего развития продукта.

## Infrastructure & DevOps practice

Личный проект: развитие собственного портфолио и эксперименты над VPS.

→ [Исходный код](https://github.com/whippedsnake/netnotes)

### Реализовано

- Hugo - static site generator.
- Форк темы Terminal, изменен стиль.
- Git submodules (тема).
- Nginx deployment.
- HTTPS через Let's Encrypt.
- VPS на Ubuntu Server.
- GitHub Actions CI/CD.
- Отдельный пользователь для деплоя.
- Минимальный харденинг сервера.


## Основные направления и профессиональные интересы

- Архитектура документации.
- Docs-as-Code процессы.
- Руководства администратора и руководства пользователя, troubleshooting-гайды.
- Документирование сложных технических продуктов.
- Работа с Linux-инфраструктурой и инструментами разработки.
